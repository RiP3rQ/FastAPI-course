## Description
REST API built using Python and the FastApi framework. Deployed on Ubuntu Linux server using DigitalOcean working on a secured website.

- **PYTHON**
- **FastAPI**
- **Docker**
- **Docker Compose**
- **Postgres**
- **JWT**
- **Postman**
- **SQLAlchemy**
- **Alembic**
- **Pytest**
- **CI/CD pipeline with GitHub Actions**
- **DigitalOcean**
- **Nginx**
- **Linux**
